
# ✈️ CSIT214 Group Project – Flight Booking Web Application

This repository contains the source code for FlyDreamAir flight booking and management system developed as part of the **CSIT214 Group Project**. The system allows users to search, book, and manage flights, as well as add services and handle payments.

---

## 📁 Project Structure

```
.
├── CSS/                     # Styling files
├── JS/                      # JavaScript logic
├── Images/                 # Image assets
├── index.html              # Entry point homepage
├── booking.html            # Flight booking page
├── change_booking.html     # Modify existing bookings
├── change_seat.html        # Change selected seats
├── add_services.html       # Add optional services
├── inflightservices.html   # View inflight services
├── payment.html            # Payment form
├── new_payment.html        # New payment page
├── new_flight.html         # Admin: Add new flight
├── managing_flight.html    # Admin: Manage flight details
├── documents.html          # Upload documents
├── guests_info.html        # Input guest details
├── select_flight.html      # Select available flights
├── seat_selection_linked.html # Advanced seat selection
├── summary.html            # Booking summary
├── update_info.html        # Update passenger info
├── searchbar.html          # Flight search logic
├── CSIT214 Group Project Gantt Chart_v3.pdf # Project timeline
```

1. Open your project folder

---

## 🛠️ Features

- ✅ **Flight Search & Booking**  
  Users can search for available flights, select preferred options, and complete the booking process.

- 🪑 **Seat Selection**  
  Users can select or change seats interactively.

- 🧳 **Add-On Services**  
  Includes meal, baggage, and inflight entertainment options.

- 💳 **Payments**  
  Securely simulate payments using `payment.html` and `new_payment.html`.

- 📄 **Document Management**  
  Upload travel documents using `documents.html`.

## 💡 Troubleshooting

| Problem                             | Solution                                                      |
|-------------------------------------|---------------------------------------------------------------|
| Website not loading properly        | Ensure all HTML, CSS, JS folders are in the same structure.  |
| CSS not applied                     | Check that file paths inside HTML are correctly linked.       |
| JS functions not working            | Open browser console (`F12`) to debug JS errors.              |
| Missing images                      | Check the `Images/` folder and ensure correct file names.     |
| Seat selection not responsive       | Refresh browser, or try using another modern browser.         |
